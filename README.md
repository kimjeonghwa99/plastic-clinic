# 성형외과 테스트 사이트 (Plastic Clinic)

성형외과 웹사이트 UI/UX 연습 및 테스트용 프로젝트입니다.  
히어로 섹션, 글래스모피즘 UI, 반응형 레이아웃을 중심으로 제작했습니다.

## 🧩 주요 기능
- 히어로 섹션 + 글래스(blur) 오버레이
- Sticky Topbar / Header
- 반응형 레이아웃 (모바일 / 태블릿 / PC)
- 상담 CTA 버튼 UI
- 카드형 정보 섹션 구성
